# 台股部位 / 風險計算機

> 先決定能虧多少，再決定買多少張。

一個純前端、單檔的台股部位大小計算機。輸入本金、單筆風險與停損，自動算出**這一筆該買幾張**、實際風險金額與風報比。所有計算都在瀏覽器完成，**資料不外傳**。

## 功能

- **風險基準倉位**：依「帳戶本金 × 單筆風險 %」反推可買股數
- **單檔資金上限**：以金額或 % 設天花板，最終張數取「風險法 vs 上限」較小值，並標示是哪個在綁手
- **停損 / 停利**：可用 **%** 或 **價位** 輸入，切換時自動換算
- **做多 / 做空** 雙向
- **零股 / 整張** 切換
- **交易成本**：內建券商手續費折扣（預設 2.8 折）與證交稅，可計入損益
- **風報比 (R)** 視覺化，集中度過高 / 超過本金會跳警示

## 使用

直接用瀏覽器打開 `index.html` 即可，無需安裝、無需後端。
